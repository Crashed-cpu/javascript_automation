How to Use
Phone Number Format:

Enter the phone number with country code, without any spaces or special characters.

Example: 911234567890 (for India)

How it Works:
  1. Enter the phone number with country code
  2. Type your message
  3. Click "Send via WhatsApp"
  4. You'll be redirected to WhatsApp Web/App with the message ready to send
Note: The recipient must have an active WhatsApp account.
