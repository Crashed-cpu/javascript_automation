How to Use
Phone Number Format:

Enter the phone number with country code, without any spaces or special characters.

Example: 14155552671 (for US/Canada)

Message Limits:
  • Standard SMS: 160 characters
  • Long SMS (concatenated): 153 characters per message
  • Unicode characters (emojis, etc.): 70 characters per message
How it Works:
  1. Enter the phone number with country code
  2. Type your message (character counter will update)
  3. Click "Send SMS"
  4. Your default SMS app will open with the message pre-filled
  5. Review and send the message
Note: Standard messaging rates may apply based on your carrier plan.
